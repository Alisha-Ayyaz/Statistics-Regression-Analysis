# Statistics-Regression-Analysis
This Statistical Regression Analysis project focuses on predictive modeling and hypothesis testing using a multivariate dataset. It covers the full analytical pipeline, beginning with Exploratory Data Analysis (EDA)and ormality testing (via P-P plots and Histograms) to ensure data integrity. The core of the project involves performing Correlation Analysis and building a Multiple Linear Regression model in SPSS to evaluate the impact of independent variables on a target outcome. By analyzing R-squared values, p-values for statistical significance, and Mean Squared Error (MSE), the project demonstrates a data-driven approach to identifying key predictors and optimizing model accuracy.
